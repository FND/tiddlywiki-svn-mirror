Portlet ccTiddly Plugin ReadMe

Used to display teamtasks content outside of tiddlywiki.

In the following example it would return a simple html list of the incomplete tasks assigned to the user Simon

http://127.0.0.1/cctiddly/Trunk/plugins/Portlet/files/portlet.php?user=Simon
