Description

Get a tiddler object from a content store

Examples

var store = $('#store').contentStore();
var tiddler = store.getTiddler('TiddlerName');

or these statements can simply be chained like this: 

var tiddler = $('#store').contentStore().getTiddler('TiddlerName');

tiddler methods - title

modifier
PhilHawksworth
modified
ddmmyyyy
changecount
0

Description

Get access to the tiddler title

Examples

Get the tiddler title value: 
tiddler.title().text();
Set the tiddler title value: 
tiddler.title().text('NewTitleValue');
A more complete example to use chained method calls to get the tiddler title value: 
var tiddlers = $('#store')
	.contentStore()
	.getTiddler('TiddlerName')
	.title()
	.text();

returns: jQuery

tiddler methods - tiddlerText

modifier
PhilHawksworth
modified
ddmmyyyy
changecount
0

Description

Get access to the tiddler text element

Examples

Get the tiddler text as html: 
tiddler.tiddlerText().html();

returns: String

Set the tiddler text value as html: 
tiddler.tiddlerText().html("<p>Some <i>new</i> tiddler text as html</p>");
Get the tiddler text as plain text 
tiddler.tiddlerText().text();

returns: String

Set the tiddler text value as plain text: 
tiddler.tiddlerText().text("Some new tiddler text as plain text");
Get the tiddler text as wiki text 
tiddler.tiddlerText().wikiText({
	format: "MediaWiki"
});

returns: String

Set the tiddler text value as plain text: 
tiddler.tiddlerText().wikiText("Some new tiddler text as plain text", {
	format: "MediaWiki"
});
