This file includes both Nested Sortable jQuery plugins, along with the test files. The NestedSortables are a set of two jQuery plugins, build on top of the Interface for jQuery, that allows you to sort and nest elements using drag and drop.

For documentation and more information please check:
http://nestedsortables.googlecode.com/
