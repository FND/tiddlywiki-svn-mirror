Basic Arrows

Creating a simple arrow in css is dead easy. All your need is a css stylesheet, two dom elements and you are ready to go. No blooming images! None of this CSS 3 magic. This makes use of the slanted borders technique which you can read up on here. Annoyingly IE (like usual) isn't playing ball. The resulting horizontal arrows when rendered in IE will not be pointed and the pointy bit will instead be flat. I'm not sure why but I'm sure there is a fix out there for this though I just haven't found it yet. Please let me know on the blog post which accompanies this experiment.

Horizontal Arrows

We construct an arrow from just 2 dom elements and head and a tail
arrow tail element
How did that work? Well we just have 4 borders of different widths and different colours. The arrow head element itself has no width/height. In the following example I've made the arrow borders different colours to explain this clearer. By careful selection of colours you can change the below to give the illusion of an arrow.
To switch the direction of the arrow just fiddle with which border has width 0px and the colours and place the arrow head dom element before the arrow tail dom element.
The left border (purple one) now has no width and instead the right border (blue one) has the big width

Vertical Arrows

These are now pretty simple to do. Now the tail becomes a div with margin-left instead of margin-top and a big height rather than big width.
